# Nexus Community Hub

Nexus is a premium community platform for developers to share projects, find jobs, and build meaningful connections.

## 🚀 Quick Start

1.  **Clone the repo**:
    ```bash
    git clone https://github.com/Ngatia259-dev/nexus-community-hub.git
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run development server**:
    ```bash
    npm run dev
    ```

## 👥 Team & Collaboration

## 🎨 Tech Stack

- **Frontend**: React (Vite)
- **Styling**: CSS Modules
- **Icons**: Lucide React
- **Routing**: React Router v6
